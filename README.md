# Sistema de Gestión de Almacén (WMS) - API Backend 🛡️📦

Este backend fue desarrollado para optimizar la gestión de inventarios y activos de la empresa **Alarmas Cobra SAC**. Es una solución robusta que integra servicios en la nube para el manejo eficiente de datos y archivos.

## 🔗 [VER CÓDIGO DEL REPOSITORIO](https://github.com/DevvAaron/Backend)

---

### 🚀 Funcionalidades y Desafíos Resueltos
Como parte del sistema de gestión para Alarmas Cobra, este backend resuelve:

1. **Gestión de Inventario Digital:** Endpoints diseñados para el control de stock, entrada y salida de equipos de seguridad.
2. **Almacenamiento en la Nube (AWS S3):** Integración con el SDK de Amazon para el almacenamiento de documentación técnica y fotos de productos, garantizando alta disponibilidad.
3. **Seguridad y Persistencia:** Manejo de datos mediante Node.js y Express con una arquitectura orientada a la mantenibilidad.

---

### 🛠️ Stack Tecnológico
* **Core:** Node.js & Express.
* **Infraestructura Cloud:** **AWS (Amazon Web Services - S3)**.
* **Gestión de Archivos:** Multer + Multer-S3.
* **API:** RESTful Architecture.
